Formação

  • Ciclo Formativo de Introdução à Lógica de Programação

    Pretalab/Olabi

    Curso livre de formação introdutória à tecnologia baseada nos pilares de conhecimento técnico, autoconhecimento e autogestão.
    Os módulos técnicos foram separados em 2, sendo eles:
    - Lógica da Programação e introdução ao JavaScript (variáveis, constantes, padrões de nomenclatura, operadores relacionais, estrutura condicional e estruturas de repetição).

    No CSS os conteúdos desenvolvidos foram: CSS inline, interno e externo, sintaxe básica, sistemas de cores, medidas absolutas, alinhamento, fontes
    identificadores, classes, espaçamentos, altura, largura, bordas, tamanho de box e margem inspecionando.
